Orientação Vocacional para Jovens

Projeto Integrador Universitário

Node.js React MongoDB Express

Contexto e Objetivo

Este projeto foi desenvolvido como parte de uma iniciativa universitária com o objetivo de auxiliar adolescentes de uma escola pública estadual de São Paulo na escolha de suas carreiras profissionais. A plataforma foi criada para fornecer orientação vocacional acessível e de qualidade para jovens que muitas vezes não têm acesso a esse tipo de serviço.

A Solução

Desenvolvemos uma plataforma web interativa que oferece testes vocacionais especialmente elaborados para o contexto dos jovens brasileiros. A plataforma não apenas identifica áreas de interesse, mas também fornece material informativo sobre diversas carreiras, mercado de trabalho e oportunidades educacionais, ajudando os estudantes a terem uma visão mais clara sobre seu futuro profissional.

Implementação

A iniciativa incluiu a apresentação da plataforma em uma escola pública estadual de São Paulo, onde os alunos puderam utilizar o sistema sob orientação da equipe do projeto. Os testes foram adaptados para considerar a realidade socioeconômica dos participantes e fornecer recomendações contextualizadas.

Características Principais

  • Teste vocacional adaptativo baseado em interesses e habilidades
  • Catálogo detalhado com mais de 100 carreiras diferentes
  • Informações sobre cursos, faculdades e opções de formação
  • Recursos educacionais para cada área de interesse
  • Interface amigável e acessível para jovens

Arquitetura Técnica

O sistema foi desenvolvido utilizando uma arquitetura de microsserviços, permitindo maior escalabilidade e manutenção simplificada. Cada componente do sistema (testes vocacionais, banco de dados de carreiras, sistema de recomendações) foi implementado como um serviço independente que se comunica através de APIs REST.

Resultados e Impacto

O projeto beneficiou mais de 200 estudantes em sua fase inicial, proporcionando orientação vocacional personalizada e material informativo que ajudou os jovens a explorarem possibilidades de carreira que muitos nem conheciam. Pesquisas de feedback após a apresentação indicaram que por volta de 70% dos participantes se sentiram mais confiantes sobre suas escolhas profissionais após utilizarem a plataforma.
