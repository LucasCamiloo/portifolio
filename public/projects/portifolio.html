Visão Geral

Este portfólio foi desenvolvido para demonstrar minhas habilidades como desenvolvedor full-stack, com foco especial em design responsivo, animações interativas e uma arquitetura modular. O site foi construído com HTML, CSS e JavaScript puros no frontend, com um backend minimalista em Node.js e Express.

Funcionalidades Principais

  • Animações interativas com CSS e JavaScript
  • Design responsivo para todos os tamanhos de tela
  • Navegação suave e indicadores de rolagem
  • Animações de entrada com efeitos de timing
  • Efeitos de glitch e sobreposição de grãos
  • Formulário de contato funcional

Detalhes Técnicos

O frontend utiliza CSS moderno com variáveis customizadas, Grid e Flexbox para layouts responsivos. As animações são desenvolvidas com CSS e JavaScript puro, garantindo uma experiência visual atraente e fluida. O backend Node.js com Express gerencia o roteamento das páginas e serve conteúdo estático de forma eficiente.

Desafios e Soluções

Um desafio significativo foi criar uma experiência visual envolvente enquanto mantendo boa performance. Para resolver isso, implementei técnicas de otimização como throttling para eventos de scroll e redimensionamento, além de técnicas de renderização eficiente para o sistema de partículas. Outro desafio foi garantir que o site funcionasse bem em todos os dispositivos, o que foi superado com um design responsivo cuidadoso e testes em múltiplos dispositivos.

Arquitetura

O projeto segue uma arquitetura simples mas eficaz, separando claramente as responsabilidades entre servidor e cliente. No frontend, o código JavaScript é modularizado por funcionalidade, facilitando a manutenção. O servidor Express foi configurado para lidar com rotas específicas para cada projeto, além de uma página 404 personalizada.

Estrutura do Projeto

O Que Aprendi

Este projeto me permitiu aprimorar meu conhecimento em animações CSS avançadas e manipulação do Canvas para criar efeitos visuais interativos. Também aprofundei minha compreensão sobre otimização de performance frontend e técnicas de design responsivo. Foi uma grande oportunidade para integrar várias tecnologias em um produto coeso e visualmente atraente.
